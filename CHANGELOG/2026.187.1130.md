# Remove the dead stage_template builder field (C10 residual)

`#Builder.stage_template` (→ `spec.Builder.StageTemplate`) is deleted. It was the
pre-C10 in-core builder stage template; C10 moved every builder's stage render into
the plugins' `kit.BuilderResolve` (OpResolve), and no builder — embedded or project
— authors `stage_template` anymore. Removing an authored-nowhere field is a codegen
refactor (every actually-authored wire key is untouched), so NO `#SchemaVersion`
bump / migration entry. A custom builder must now be an `external_builder` plugin.
