# Agent resolve wire types (Cutover E)

Support the agent de-type: candy/plugin-agent resolves the AI-CLI catalog into a
generic AgentExecSpec the kernel's check/iterate harness consumes without importing
the concrete `agent` kind.

- `spec/agent_wire.go` (hand-written): `AgentResolveInput` (opaque agent bodies +
  selected name) → `AgentResolveReply{ Spec *AgentExecSpec; Name }`. `AgentExecSpec`
  is a kind-agnostic "launch + version-capture + iterate-poll a long-running CLI"
  descriptor (command/prompt_via/version_command/timeout/env/working_dir/credential/
  progress_*/output_format) — the resolve-to-envelope form of an agent, defaults
  applied.

No CUE-schema change (the authored `#Agent` shape is untouched) — no `#SchemaVersion`
bump, no gen-repro drift.
