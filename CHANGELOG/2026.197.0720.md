# 2026.197.0720 — VM disk-build resolve envelope (P8b-rest)

Adds `#VmBuildRequest.force` (restoring the `--force` flag that was dropped from
this seam when `charly vm build` externalized to `command:vm` in P10 — a
pre-existing gap, load-bearing for the disk-build engine's freshness-skip
check) and enriches `#VmBuildReply` from empty to carry everything
`candy/plugin-vm` needs to run the VM-disk build engine itself: the resolved
`#Vm`-shaped entity (`vm_json`, opaque — the same convention
`#ConfigResolveReply.vm_json` already uses for a `#Vm`-shaped payload), the
matched distro/builder for a `bootstrap` source (`distro_json`/`builder_json`,
opaque hand-written runtime types), the resolved build engine + rootful mode,
pre-resolved bootc/builder image refs, output paths, and the entity's existing
`#VmDeployState` (already a CUE def — a typed embed via `@go(...,
type=*VmDeployState)`, matching the established pattern
`#ConfigResolveReply.vm_state` already uses for the same type).

This is the sdk leg of P8b-rest: `charly vm build`'s disk-build ENGINE
(privileged pacstrap/bootc-install/qemu-img exec, cloud-init render) moves
from `charly/` core into `candy/plugin-vm` — the same inversion
`candy/plugin-build`'s podman drive already went through behind
`HostBuild("build-prep")` in P8b. The host-builder ("vm-build") shrinks to
PREP+RESOLVE only.

Also fixes a stale code-comment divergence found during this cutover's R1
sweep: `deploy_wire.go`'s `BuildRequest` doc comment named the host-builder
`HostBuild("build-resolve", ...)` — the seam was renamed to `"build-prep"` in
an earlier P8b wave but this one comment was missed. Corrected to match the
live registration in `charly/build_resolve_host.go`.

Round-1 pr-validator review caught the same stale `"build-resolve"` name still
present in two `schema/buildresolve.cue` doc comments (the claim-keyed R5 sweep
had only checked `deploy_wire.go`) — fixed in a follow-up commit; the repo-wide
`git grep '"build-resolve"'` self-test (excluding `CHANGELOG/`) now genuinely
returns zero hits.
