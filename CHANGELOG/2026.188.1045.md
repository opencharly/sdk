# Add the OpenCharly PR template

Adds `.github/pull_request_template.md` so PRs to this repo prompt for the full
description the fresh `pr-validator` agent requires (summary · how-tested/R10
evidence · attribution tier · R0–R10 checklist). Part of the project-wide PR-only,
agent-validated landing policy (a direct push to `main` is forbidden and
mechanically disabled; every change lands through a validated PR). Documentation-only.
